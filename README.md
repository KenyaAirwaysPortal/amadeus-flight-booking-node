## Set up Node server

Go to the directory

```sh
cd server
```
Install depedencies
```sh
npm install
```
Launch the server
```sh
npm start
```

## Set up Vue js (client)

You need to install vue js cli 


```sh
npm install -g @vue/cli
# OR
yarn global add @vue/cli
```

```sh
cd client
```
Install depedencies
```sh
npm install
```
Serve the project

```sh
npm run serve
```
