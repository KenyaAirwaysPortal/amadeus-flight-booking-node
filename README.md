## Set up Node server

Go to the directory

```sh
cd server
```
Install depedencies
```sh
npm install
```
Launch the server
```sh
npm start
```

## Set up Vue js 

You need to install vue js cli 


```sh
npm install -g @vue/cli
# OR
yarn global add @vue/cli
```

```sh
cd VueJs
```
Install depedencies
```sh
npm i
```
Serve the project

```sh
npm run serve
```
